## Description

**Binomial codes** encode a logical qubit into a finite superposition of Fock states of a single bosonic mode, with amplitudes set by square roots of binomial coefficients. In circuit QED, the mode is typically a long-lived microwave cavity dispersively coupled to a transmon ancilla.

The simplest single-loss-correcting example is the **0-2-4 code**,

$$|0_L\rangle = \frac{|0\rangle + |4\rangle}{\sqrt{2}}, \qquad |1_L\rangle = |2\rangle,$$

which stores a logical qubit entirely in the even-parity Fock subspace. A single photon loss maps the state into an odd-parity error space that can be detected without directly measuring the logical information.

Compared with cat codes, binomial codes use **finite-support, exactly orthogonal** code words rather than coherent-state superpositions. That makes them attractive as hardware-efficient discrete-variable bosonic encodings for cavity-based quantum error correction, logical control, and modular networking proposals.

## Hamiltonian

For the bosonic mode itself,

$$H_0 = \hbar \omega_c a^\dagger a,$$

with annihilation operator $a$ and cavity frequency $\omega_c$. In the superconducting cavity-transmon setting, a standard effective control model is the dispersive Hamiltonian

$$H_\mathrm{disp}/\hbar = \omega_c a^\dagger a + \frac{\omega_q}{2}\sigma_z + \chi a^\dagger a\,\sigma_z,$$

where the ancilla transmon enables state preparation, syndrome extraction, and conditional recovery.

The dominant noise channel is photon loss,

$$L_\mathrm{loss} = \sqrt{\kappa}\,a,$$

with possible additional dephasing generated by powers of $\hat n = a^\dagger a$. Binomial-code words are chosen so that the Knill-Laflamme conditions hold for a target set of bosonic errors,

$$\langle W_\mu|E_\ell^\dagger E_m|W_\nu\rangle = C_{\ell m}\,\delta_{\mu\nu},$$

for the correctable operators $E_m$. Error detection is commonly phrased through a generalized parity or discrete rotation measurement,

$$\Pi_{S+1} = e^{i 2\pi \hat n/(S+1)},$$

which distinguishes the code space from loss-shifted error spaces. For the 0-2-4 code, ordinary photon-number parity $(-1)^{\hat n}$ is sufficient to detect a single loss event.

## Motivation

- Gives bosonic quantum error correction with a **single oscillator mode plus ancilla**, rather than a large multi-qubit block.
- Targets the dominant cavity error channel, **photon loss**, with syndrome measurements that are natural in circuit QED.
- Keeps code words **exactly orthogonal** and finitely supported in Fock space.
- Supports hardware-efficient proposals for logical gates, state transfer, and modular bosonic networking.

## Experimental Status

**Original proposal, Michael et al. (2016):**
- Introduced the binomial-code family for correcting combinations of boson loss, boson gain, and dephasing in a single mode.

**First direct binomial logical-qubit experiment, Hu et al. (2019):**
- Demonstrated quantum error correction and a universal logical gate set on a **binomial bosonic logical qubit** in a superconducting cavity platform.
- Reported **97.0% average logical-gate process fidelity** and a **2.8× lifetime improvement** over the uncorrected encoding in the corresponding preprint/journal result.
- This is the cleanest experimental milestone for the specific binomial-code encoding, and it supersedes the older habit of informally crediting related cat-code papers here.

**Beyond break-even discrete-variable bosonic QEC, Ni et al. (2023):**
- Demonstrated a **binomially encoded** microwave-cavity logical qubit with real-time feedback QEC.
- Reported about **16% lifetime enhancement beyond break-even**, making this the clearest currently verified experimental benchmark for binomial-code QEC in the literature reviewed here.

**Recent 2024-2026 direction:**
- Recent work is concentrated on improved logical control and state preparation, for example ancilla-free single-qubit rotations and noise-resilient geometric-gate proposals.
- In this audit I did **not** find a clearly established post-2023 experimental lifetime milestone that should replace Ni et al. as the conservative binomial-code benchmark.
