Introduces subsystem codes and the Bacon-Shor code family. Shows that by introducing gauge degrees of freedom, the weight of syndrome measurements can be reduced to 2 while maintaining the code distance, dramatically simplifying hardware requirements for quantum error correction.

## Key Results
- Subsystem code framework: logical + gauge + stabilizer decomposition
- [[m²,1,m]] Bacon-Shor code on m×m grid
- Syndrome extraction via weight-2 gauge measurements only
- Can be made asymmetric (m₁×m₂) for biased noise
