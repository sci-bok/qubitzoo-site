## Summary

Generalizes the cluster-state model of quantum computation from discrete to continuous variables. Shows that CV cluster states combined with homodyne detection enable arbitrary multimode Gaussian transformations. Universality requires any single non-Gaussian measurement. Proposes an optical implementation using squeezed-light sources, linear optics, and homodyne detection.
