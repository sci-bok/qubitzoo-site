Introduces color codes: a family of topological quantum error-correcting codes defined on trivalent, 3-colorable lattices. Color codes support transversal implementation of the full Clifford group, a key advantage over surface codes which only support transversal CNOT.

## Key Results
- Transversal Clifford gates (H, S, CNOT) — surface code only has transversal CNOT
- Defined on 2D triangular/hexagonal lattices with 3-colorability
- Distance-3 code: [[7,1,3]] (Steane code is a special case)
- Topological protection equivalent to surface code
