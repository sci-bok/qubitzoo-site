Comprehensive roadmap for implementing the surface code on superconducting hardware. Establishes the practical resource estimates that have guided the field's hardware targets for a decade: ~1% physical error rate threshold, ~1000 physical qubits per logical qubit for useful computations.

## Key Results
- Circuit-level threshold: ~1% per gate/measurement
- Resource estimates for Shor's algorithm: ~20M physical qubits
- Lattice surgery for logical operations
- Concrete superconducting hardware requirements
