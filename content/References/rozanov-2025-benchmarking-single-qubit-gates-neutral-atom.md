Benchmarks native single-qubit performance on a neutral-atom processor using direct randomized benchmarking and gate-set tomography.

## Key Results
- Reports average single-qubit gate fidelity of 99.963% using direct randomized benchmarking.
- Extends the protocol to a 25-qubit neutral-atom array under global single-qubit control.
- Uses gate-set tomography plus gauge optimization to obtain SPAM-robust gate characterization.
- Provides an independent, methodology-focused confirmation that neutral-atom 1Q control is now in the surface-code-threshold regime.
