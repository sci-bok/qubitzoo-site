## Summary

Generates a large-scale two-dimensional continuous-variable cluster state with a $5 \times 1240 = 6{,}200$ mode square lattice structure using time-domain multiplexing. The 2D lattice structure is compatible with topological error-correcting codes for fault-tolerant quantum computation at sufficient squeezing levels. Demonstrates the scalability of CV entanglement generation in a compact tabletop setup.
