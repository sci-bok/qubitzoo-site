Reports 4D surface-code experiments on Quantinuum's H2 QCCD quantum computer, configured with 56 physical $^{171}$Yb$^+$ ion qubits, and summarizes the physical-layer error budget relevant to logical-memory performance.

## Key Results
- H2 configuration used for this work contained 56 physical $^{171}$Yb$^+$ qubits
- Physical-layer error budget was dominated by two-qubit gate errors $\le 1.3\times10^{-3}$, single-qubit gate errors $\sim 3\times10^{-5}$, and SPAM errors $\sim 1.5\times10^{-3}$
- Demonstrated the first experimental single-shot quantum error correction with the [[33,1,4]] 4D surface code on a QCCD trapped-ion processor
